# Agentic Astro

Live workshop repository — solving astronomy problems with AI agents.

## Context

This repo is used during a live session. Code is written in real time with
audience input. Prioritize working solutions over polish. Astronomy domain:
expect FITS files, astropy, numpy, matplotlib, catalog cross-matching, coordinate
systems, radio interferometry tools (casacore, wsclean, etc.).

## Conventions

- Python 3.11+
- Use astropy for astronomy primitives (coordinates, units, FITS I/O)
- Prefer numpy/scipy for numerical work
- Matplotlib for quick plots; keep visualizations simple and readable
- Each problem tackled during the session gets its own directory
