# Agentic Astro

**Live workshop: Solving Astronomy Problems with AI Agents**

**Date:** Thursday, June 25, 2026

**Host:** Gijs Molenaar

## About

A hands-on live session where we use AI coding agents to tackle real astronomy
problems in real time. No slides, no pre-baked demos — just problems, ideas, and
an agent in the loop. We'll pick challenges from the audience and the backlog,
pair with an AI agent, and see how far we get before the clock runs out.

Topics may include:

- Analyzing astronomical survey data
- Building pipelines for radio interferometry calibration
- Wrangling FITS files, catalogs, and coordinate transforms
- Automating repetitive observatory workflows
- Prototyping visualization and analysis tools from scratch

## What to expect

- Live problem-solving, not a polished lecture
- Real failures and debugging alongside the wins
- A front-row seat to agentic coding workflows applied to science
- Audience participation — bring your gnarliest astro-coding problem

## Prerequisites

Curiosity. Familiarity with Python and basic astronomy concepts helps but is not
required.

## Repository

This repo will collect the code, notebooks, and artifacts produced during the
